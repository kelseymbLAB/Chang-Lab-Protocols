Hello! Welcome to the file storage repository for the RNA-Seq to Contextualization Pipeline. 

This repository contains the files I used to generate my end model, as well as the end model I used. You can generate the final model and all important files through using the pipeline, but if you just want to work with the contextualized model it is there.

Walkthrough is here: https://github.com/kelseymbLAB/Chang-Lab-Protocols/wiki/FASTQ-to-Contextualization-Pipeline

Final Contextualized Model: RPMI_GIMME_Model.xml

Alignment Files: RNA-Seq data downloaded from SRA or your sequencing company, your genomic.gtf and .fna from your reference genome. In my case, its genomic.gtf and GCF_000026505.1_ASM2650v1_genomic.fna for LGG. Finally, the align.sbatch script.
